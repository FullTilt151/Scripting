ListAll.vbs - returns a list of all available BIOS settings and their current values on a local system. 
ListAllRemote.vbs - returns a list of all available BIOS settings and their current values on the specified remote system. 
SetConfig.vbs - sets the specified BIOS setting to the specified value on a local system. 
SetConfigPassword.vbs - sets the specified BIOS setting to the specified value on a local system, where there exists a BIOS administrator password. 
SetConfigRemote.vbs - sets the specified BIOS setting to the specified value on the specified remote system 
SetConfigPasswordRemote.vbs - sets the specified BIOS setting to the specified value on the specified remote system, where there exists a BIOS administrator password.  
LoadDefaults.vbs - loads BIOS default settings on a local system. 
SetAdminPassword.vbs - Update Administrator password on a local system
